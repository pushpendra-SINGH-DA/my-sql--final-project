## TRANSLATOR :
                as machine can only understand low level lang (binary lang) so we need a translator which can convert our high level code into low level code.

                                   translator
HLL ----------------------------------------------------------------> LLL
source code                                                             machine code 
                                               
(** a line of code is called statement in python.)

TYPES OF TRANSLATOR :
1) interpreter :
                  it convert our source code written in hll into machine code (binary code), line by line (statement by statement)
                  if and only if there is no error in our statement .
  ---> does not create a separate . exe file but instead 
  ----> translated code directly send to cpu for processing . (one the fly )
* RUN TIME ERROR   
2) compiler :
             it convert our source code written in hll into machine code (binary code)in one go ,(whole file at once)
                                                                                                      (whole project at once) .

if and only if there is no syntax error in our source code (files).
----> create a separate ,exe file which is runnable.
* SYNTEX ERROR
